AlexBot - Commands

AlexBot's Prefix is "lol"

Roblox Related Commands

  • lol uid {Roblox Username}. | Returns the Roblox Userid of the person with the username specified
  • lol rbxusername {@mention|Discord Userid} {"mention"|"id"} | This returns the Roblox Username and Userid from someone's Discord Userid or mention/ping. At the end, you type "mention" or "id" to select what type of argument you are putting in

Discord Related Commands

  • lol spamping {@user} {times} {optional:message} | It is obvious what this does
  • lol spam {times} {message} | Spams your message the amount of times you specify

General Bot Commands

  • lol latency | This returns the current latency in seconds
  • lol helpme | Gives you help
  • lol suggest {suggestion} | Suggest features :) Notice: Currently Disabled
  • lol info | Information about the bot

Fun Commands

  • lol iss | This tells you the position of the International Space Station
  • lol numberinfo {number} | Gives you facts about the number you input
  • lol github {repository name} | Searches github for a repository with the name you provide
  • lol botkahoot {game pin} | Puts bots in the Kahoot game of the game pin you provide

Actually Useful Commands

  • lol bitcoin | Gets the price of a Bitcoin from coinbase api
